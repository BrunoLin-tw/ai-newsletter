# 🤖 AI Daily Newsletter - 使用指南

## 📋 頁面說明

### 1. **最新頁面** (`index.html`)
- 顯示**最近 7-10 天**的電子報內容
- 使用**卡片式設計**，點擊展開/收起查看詳情
- **手機優化**：內容預設收起，節省空間

### 2. **檔案頁面** (`archive.html`)  
- 按**月份分組**展示所有歷史電子報
- 每個月份可**摺疊/展開**
- 適合尋找**特定月份**的內容

### 3. **搜尋頁面** (`search.html`)
- **實時搜尋**功能，輸入關鍵字即可搜尋
- 支援搜尋**標題、日期、內容**
- 搜尋詞會**高亮顯示**

---

## 📱 手機優化特性

### ✅ 動態摺疊
- 卡片預設收起，只顯示標題和日期
- 點擊即展開查看完整內容
- **大幅減少頁面高度**，提升閱讀體驗

### ✅ 響應式設計
- **768px 以下**：調整排版和字體
- **480px 以下**：進一步優化小螢幕顯示
- 保持導航欄粘性（sticky），方便切換頁面

### ✅ 快速導航
- 頂部粘性導航欄
- 三個頁面間快速切換
- 清晰的頁面指示（active 狀態）
