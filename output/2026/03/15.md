主要分類：🔧 硬核技術更新

🧪 Microsoft Research 開源 AgentRx，替 AI agents 補上「失敗診斷層」 — 這套 framework 不只是再做一次 prompt，而是從 tool schema 與 domain policy 自動生成可執行約束，逐步找出第一個不可挽回的 critical failure step。官方 benchmark 顯示，失敗定位準確率提升 23.6%、根因歸因提升 22.9%，對要把 agent 拉進真實工作流的團隊很有參考價值。
🔗 [來源](https://www.microsoft.com/en-us/research/blog/systematic-debugging-for-ai-agents-introducing-the-agentrx-framework/)

⚡ Microsoft Foundry 接上 Fireworks AI，把開放模型推論直接拉進 Azure 企業控制平面 — 這次重點不只是多一個 model provider，而是把 DeepSeek V3.2、gpt-oss-120b、Kimi K2.5、MiniMax M2.5 等開放模型，整合到同一個企業級評估、治理與部署環境。Fireworks 官方引擎已號稱每天處理逾 13 兆 token，對企業來說，開放模型的「上線難度」正在被明顯降低。
🔗 [來源](https://azure.microsoft.com/en-us/blog/introducing-fireworks-ai-on-microsoft-foundry-bringing-high-performance-low-latency-open-model-inference-to-azure/)

🏗️ Anthropic 宣布投資 500 億美元建美國 AI 基礎設施，資料中心先落德州與紐約 — 與其說這是單一建案，不如說是 frontier model 公司的算力內建化。Anthropic 強調站點將在 2026 年陸續上線，並由 Fluidstack 協助快速交付 gigawatt 級電力，這代表模型競爭正在從 API 與模型能力，繼續往電力、機櫃與交付速度下沉。
🔗 [來源](https://www.anthropic.com/news/anthropic-invests-50-billion-in-american-ai-infrastructure)

🏢 大廠動態

🏭 OpenAI 啟動美國 AI 供應鏈製造 RFP，從晶片機房一路延伸到機器人零組件 — 這次不是單純喊口號，而是正式向美國本土製造商徵求 proposal，涵蓋資料中心硬體、散熱、供電、測試組裝與消費電子模組。看點在於 OpenAI 已把 AI 基建視為更廣義的產業鏈重組，不再只盯 GPU 與雲端租用。
🔗 [來源](https://openai.com/index/strengthening-the-us-ai-supply-chain/)

🎤 NVIDIA GTC 2026 進入倒數，議程焦點明擺著：agentic AI、physical AI 與 AI infrastructure — NVIDIA 最新預熱文把今年主軸講得很直白：從 token generation、電力散熱，到 open models、機器人與數位分身，整個 AI 產業鏈都在往更重的基建與更實際的部署場景走。Jensen Huang 這場 keynote 幾乎已成下週產業風向球。
🔗 [來源](https://blogs.nvidia.com/blog/gtc-2026-news/)

🎮 有趣的小工具應用

🩺 Microsoft 推出 Copilot Health，要把健康紀錄、穿戴裝置與就醫資訊整成可對話的個人醫療助手 — 它主打的不只是問答，而是能串接超過 50 種 wearable、5 萬家美國醫療院所資料，以及 Function 檢測結果，幫使用者整理病史脈絡與就診問題清單。這類產品若真能把「資料很多但看不懂」變成可行動 insight，會是消費級 AI 少數真正高頻又高黏著的場景。
🔗 [來源](https://microsoft.ai/news/introducing-copilot-health/)
