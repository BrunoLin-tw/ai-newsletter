主要分類：🔧 硬核技術更新

🧠 Cursor 推出 Automations，讓程式代理從「有人盯場」走向常駐流程化 — 新功能可依 code commit、Slack 訊息或定時器自動啟動代理，連 PagerDuty 事件也能直接觸發 log 查詢與安全審查。這代表 agentic coding 正從單次 prompt 工具，進化成可持續運轉的工程流水線。  
🔗 [來源](https://techcrunch.com/2026/03/05/cursor-is-rolling-out-a-new-system-for-agentic-coding/)

🎬 Luma 發布 Unified Intelligence 模型家族，直接把多模態生成包成創意代理 — 其 Uni-1 模型以單一推理系統處理文字、圖片、影音與空間理解，並能協同 Veo 3、ElevenLabs 等模型完成端到端廣告素材生成。對品牌與代理商來說，AI 的價值開始從「生一張圖」升級成「交付整套 campaign」。  
🔗 [來源](https://techcrunch.com/2026/03/05/exclusive-luma-launches-creative-ai-agents-powered-by-its-new-unified-intelligence-models/)

🏢 大廠動態

🏥 AWS 推出 Amazon Connect Health，把 AI agent 正式帶進醫療行政流程 — 平台主打 HIPAA 合規，能串接 EHR 系統，自動處理病患身分驗證、環境式紀錄、排程與後續醫療編碼。關鍵不只是模型能力，而是 AWS 把 agent 包進可落地的醫療工作流與計價機制，瞄準的是醫療體系最痛的行政成本。  
🔗 [來源](https://techcrunch.com/2026/03/05/aws-amazon-connect-health-ai-agent-platform-health-care-providers/)

💬 Meta 在歐洲開放 WhatsApp 給第三方 AI 聊天機器人上架，但改用按訊息收費 — 為回應歐盟反壟斷壓力，Meta 允許一般型 AI chatbot 透過 WhatsApp Business API 提供服務，費率約每則 0.049 至 0.1323 歐元。這是平台治理與商業模式一起鬆動的訊號：AI 助理未來不只拚模型，還要拚分發通路與單位經濟。  
🔗 [來源](https://techcrunch.com/2026/03/05/meta-will-allow-rival-ai-chatbots-on-whatsapp-in-europe-but-for-a-fee/)

🔎 Google 將 Gemini Canvas 擴大到 Search 的 AI Mode 全美英文用戶 — 使用者已可直接在搜尋中把研究資料整理成文件、原型 app、互動小工具或寫作草稿，背後結合 Gemini 與 Google Knowledge Graph。這一步很重要，因為 Google 正把生成式 AI 從獨立助手，往搜尋主入口原生整合。  
🔗 [來源](https://techcrunch.com/2026/03/04/https-techcrunch-com-2026-03-04-google-search-rolls-out-geminis-canvas-in-ai-mode-to-all-us-users/)
