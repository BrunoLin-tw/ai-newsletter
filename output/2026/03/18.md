主要分類：🔧 硬核技術更新

🧠 OpenAI 發布 GPT‑5.4 mini / nano：小模型也要「能打又便宜」— 針對高吞吐場景把延遲與成本打下來
GPT‑5.4 mini 主打 2x 以上速度、400k context，並在 SWE‑Bench Pro、OSWorld‑Verified 等評測逼近大模型；nano 則是更小更便宜的版本，適合分類/抽取/排序與「coding subagent」這類支援型工作負載。價格也直接公開（mini：$0.75/M input、$4.50/M output；nano：$0.20/M input、$1.25/M output）。
🔗 [來源](https://openai.com/index/introducing-gpt-5-4-mini-and-nano/)

🧩 Google Gemini API 強化 agent 工具鏈：同一個 request 內建工具 + 自訂 function 一次打包
Google 讓開發者在單次請求裡同時帶入 Google Search/Maps 等內建工具與自家 function calling，並提供「context circulation」把各工具回應完整保留在模型上下文中，減少你在外層寫一堆 orchestration glue code 的痛苦；同時也把 Maps grounding 擴到 Gemini 3 家族，並建議用 Interactions API 取得 server-side state 管理與 reasoning traces。
🔗 [來源](https://blog.google/innovation-and-ai/technology/developers-tools/gemini-api-tooling-updates/)

⚙️ OpenAI 推出「Parameter Golf」研究挑戰：16MB 成品 + 10 分鐘訓練，逼你把預訓練效率玩到極限
目標是在固定 FineWeb 資料集上、在「權重+訓練碼」總共 16MB、且 8×H100 只跑 10 分鐘的硬限制下，最小化 held-out loss。這種極端約束很像把壓縮、訓練技巧、架構取捨一次攤開比賽，也很可能帶出更實用的「小模型高效預訓練」套路。
🔗 [來源](https://openai.com/index/parameter-golf/)

🗣️ Anthropic 公布 8 萬人 AI 訪談：用 Claude 當「AI 訪談員」收集 159 國、70 語言的希望與焦慮
Anthropic 讓 Claude.ai 使用者與「Anthropic Interviewer」對話，蒐集 80,508 份開放式訪談，再用 Claude-based classifiers 把人們最想要的 AI 角色分群（例如：工作更卓越、個人成長、生活管理、時間自由、財務獨立…），並做出可篩選的 Quote Wall。這類大規模質化資料，對產品設計與 AI 治理討論都很有參考價值。
🔗 [來源](https://www.anthropic.com/features/81k-interviews)

🎮 有趣的小工具應用

🎨 Google Labs Stitch 主打「vibe design」：把自然語言直接變高保真 UI，還加了設計 agent 與 Agent manager
Stitch 進化成 AI-native 的設計畫布（infinite canvas），讓你用文字/圖片/程式碼當 context 快速迭代 UI；新增設計 agent 能跨整個專案歷程推理，並用 Agent manager 同時追多條設計分支。還推出 DESIGN.md 讓設計規則可匯出/匯入，搭配 MCP server/SDK 串接到開發工具鏈，從設計一路走到 code 更順。
