主要分類：🔧 硬核技術更新

🧠 NVIDIA 推出 Nemotron 3 Super，瞄準多代理 AI 的效能瓶頸 — 這款 120B 開放模型採 hybrid MoE 架構、僅啟用 12B active parameters，支援 100 萬 token context，官方宣稱吞吐量最高提升 5 倍、準確率提升 2 倍。對 agent workflow 來說，重點不是只看模型大不大，而是能不能把長上下文、工具調用與多步推理做得夠快。
🔗 [來源](https://blogs.nvidia.com/blog/nemotron-3-super-agentic-ai/)

🛡️ OpenAI 公開 AI agent 連結安全機制，直接對準 prompt injection 外洩風險 — 新方案核心不是只信任「網站網域」，而是先用獨立 web index 驗證某個 URL 是否曾在公開網路上出現，再決定能否自動抓取。這是在 agent 真正開始替人「點連結、看頁面」後，很關鍵的一層防資料外洩設計。
🔗 [來源](https://openai.com/index/ai-agent-link-safety/)

🏢 大廠動態

💼 Anthropic 砸 1 億美元啟動 Claude Partner Network — 這不是單純招募經銷商，而是把 training、技術支援、共同銷售與認證制度一起包進去，還同步推出 Claude Certified Architect, Foundations。訊號很明確：Anthropic 正把 Claude 從模型供應商，往企業導入生態系經營者推進。
🔗 [來源](https://www.anthropic.com/news/claude-partner-network)

👔 Adobe 宣布 CEO 交棒，市場把壓力直接對準 AI 轉型 — 掌舵 18 年的 Shantanu Narayen 將在接班人確定後卸任，消息公布後股價盤後下跌逾 7%。這件事值得看，不只是人事，而是投資人已開始用更高標準檢視創意軟體龍頭在生成式 AI 時代的產品與成長敘事。
🔗 [來源](https://finance.yahoo.com/news/adobe-announces-ceo-transition-shares-202025546.html)

🎮 有趣的小工具應用

🗺️ Google 把 Gemini 真正塞進 Maps，讓地圖開始能「聊天」 — 新功能 Ask Maps 可直接回答像是「附近哪裡能充電又不用排隊買咖啡」這種情境問題，並結合 3D 路況與更自然的導航提示。Google 想做的已不是搜尋地點，而是把地圖升級成即時的生活決策介面。
🔗 [來源](https://blog.google/products-and-platforms/products/maps/ask-maps-immersive-navigation/)

🎬 Webflow 收購 Vidoso，往 agentic marketing 再跨一步 — Vidoso 能把演講、影片與長內容自動轉成短片、部落格與社群素材，補上品牌內容生成到投放分析之間的流程斷點。比起單點生成工具，這筆交易更像是在押注「品牌規則 + 工作流 + 生成 AI」的一體化行銷平台。
🔗 [來源](https://techcrunch.com/2026/03/12/webflow-buys-ai-content-generation-platform-vidoso-to-bolster-its-marketing-suite/)
