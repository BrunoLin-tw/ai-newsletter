**🔧 硬核技術更新**

🧠 OpenAI GPT-5.5 正式發布：號稱史上最聰明且最直觀的模型，主打更快的 agentic coding、更強的 computer use 以及更流暢的知識工作。支援 code interpreter、hosted shell、apply patch、skills、computer use、MCP、tool search 等完整工具鏈，已在 ChatGPT 與 Codex 中開放付費訂閱使用。
🔗 [來源](https://openai.com/index/introducing-gpt-5-5/)

🧠 DeepSeek V4 開放預覽：4 月 24 日發布的旗艦模型，支援 1M context 與雙模式（Thinking/Non-Thinking）。Pro 版本 1.6T 參數（49B 激活），Flash 版本 284B 參數（13B 激活），以極低價格提供接近前沿模型效能，被評為「幾乎達到前沿水準，價格卻只是零頭」。
🔗 [來源](https://www.technologyreview.com/2026/04/24/1136422/why-deepseeks-v4-matters/)

**🏢 大廠最新動態**

💰 Google 宣布向 Anthropic 追加投資最多 400 億美元：這是繼雙方既有合作後的最大手筆，目標強化 Claude Code 產品線的運算基礎設施。此舉顯示 Google 在 AI 投資策略上的矛盾——既是投資者也是競爭對手。

💰 Anthropic Claude Mythos 引發全球資安擔憂：Mythos 能自主發現並利用軟體漏洞，甚至可獨立找出並武器化 CVE-2026-4747（FreeBSD NFS 17 年遠端程式碼執行漏洞）。World Economic Forum 警告這類模型降低攻擊門檻，讓初階駭客也能發動複雜攻擊。英國監管機構已開始評估其對金融業的風險。
🔗 [來源](https://www.weforum.org/stories/2026/04/anthropic-mythos-ai-cybersecurity/)

🔐 Apple 的 vibe coding 應用程式策略挨轟：Replit、Anything 等新創投訴 Apple 對 iOS App Store 規則執行不一致，導致 vibe coding 應用程式頻頻被拒或下架，開發者怨聲載道。
