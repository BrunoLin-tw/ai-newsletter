**🔧 硬核技術更新**

🧠 **TorchTPU 讓 PyTorch 幾乎原生跑上 Google TPU** — Google 公開 TorchTPU 架構，主打把既有 PyTorch workload 用最少改動搬上 TPU，從 debug eager、strict eager 到 fused eager 都包進來，還直接接上 torch.compile 與 XLA。這條夠硬，因為它不是又一個包裝層，而是在解「PyTorch 生態怎麼吃到 TPU 超算規模」這個老痛點。
🔗 [來源](https://developers.googleblog.com/en/torchtpu-running-pytorch-natively-on-tpus-at-google-scale/)

🧩 **Google ADK 把 agent skills 做成可按需載入的能力模組** — Google 在 ADK 指南裡把 SkillToolset 講清楚了，從 L1 metadata、L2 instructions 到 L3 references 分層載入，目標是把原本塞爆 system prompt 的知識改成按需揭露。更兇的是最後直接走到 skill factory，讓 agent 能在執行時自己寫出新技能。
🔗 [來源](https://developers.googleblog.com/en/developers-guide-to-building-adk-agents-with-skills/)

**🏢 大廠動態**

🏗️ **Microsoft 砸逾 10 億美元布局泰國 AI 基建** — Microsoft 宣布 2026 到 2028 將在泰國投入超過 10 億美元的雲端與 AI 基礎設施，同步推進公部門 AI 治理對話、電商應用可行性研究與數位人才培訓。這不是單點投資，而是想把泰國直接推成東南亞 AI 落地跳板。
🔗 [來源](https://news.microsoft.com/source/asia/2026/04/10/microsoft-charts-ai-powered-path-forward-in-thailand/)

🔐 **OpenAI 因 Axios 供應鏈事件輪替 macOS 簽章憑證** — OpenAI 說明，macOS app signing workflow 曾抓到遭污染的 Axios 版本，雖然目前沒有用戶資料外洩或軟體被竄改的證據，還是決定撤銷並輪替簽章憑證。ChatGPT Desktop、Codex App、Codex CLI 與 Atlas 的舊版 app 之後都得升級，這種供應鏈事件現在連 AI 桌面工具都躲不掉。
🔗 [來源](https://openai.com/index/axios-developer-tool-compromise/)

🌍 **南非端出國家 AI 政策草案，連監管機構與算力都一起規劃** — 南非發布 draft national AI policy，提議設立 National AI Commission、AI Ethics Board 與 AI Regulatory Authority，還要用稅務優惠、補助和超算投資拉動本地 AI 生態。這條值得看，因為它不是只談原則，而是把治理架構、產業誘因和主權算力一次端上桌。
🔗 [來源](https://enterpriseai.economictimes.indiatimes.com/news/industry/south-africa-launches-draft-ai-policy-to-become-continental-leader/130188424)

🤖 **GitHub Copilot 在 VS Code 開始把 Autopilot 推進公開預覽** — GitHub 這波 VS Code 更新把 agent 自主權拉高了，新增 Autopilot、browser debugging、圖片與影片 chat 支援，還把 chat customizations 收進同一個管理介面。簡單講，就是 Copilot 不只幫你補字，開始更像一個能自己跑一段流程的 coding agent。

🔗 [來源](https://aws.amazon.com/blogs/aws/aws-weekly-roundup-aws-devops-agent-security-agent-ga-product-lifecycle-updates-and-more-april-6-2026/)
